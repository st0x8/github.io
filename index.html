数学只是一个工具

发表于 2016-05-07   |   分类于 Mathematics

数学是人类发明的工具。这个工具有使用价值，为了让它能更好地发挥作用，先人们不断对它进行补充、完善，发展成了如今这个庞然大物。

数学是一个伟大的发明，凝聚了很多人的心血。但它只是一个工具，我认为不应该对它生出盲目的畏惧，要用平常心去了解它，学习如何正确地使用它，就像我们学习使用家用电器那样，最终让它发挥作用。

以上想法是我在思考除数为什么不能为0时冒出的。除数为什么不能为0呢？这个规定完全是人为添加的，目的就是为了合乎逻辑。除法是乘法的逆运算，就是已知两个因数的积与其中一个因数，求另一个因数的运算。如果一个因数是0，积必定为0。如果被除数不为0，除数为0，就是说－－我有一个因数为0，积却不为0，这就矛盾了，所以干脆限定除数不能为0.

中学数学笔记

发表于 2016-04-20   |   分类于 Mathematics

有理数

数

  • 自然数的个数是无限多的，对于任何一个自然数来说，总还有比它更大的自然数
  • 0不是自然数
  • 自然数和0都是整数
  • 真分数：分子比分母小的分数
  • 假分数：分子大于或等于分母的分数
  • 0既不是正数也不是负数
  • 正整数、零与负整数都是整数

运算

  • 四则运算：加法、减法、乘法、除法
  • 被乘数 × 乘数 ＝ 积 或 因数甲 × 因数乙 ＝ 积。任意两个数，总可以相乘，求出它们的积。
  • 除法是已知两个因数的积与其中一个因数求另一个因数的运算。已知的积叫做被除数，已知的一个因数叫做除数，所求的另一个因数叫做商。
    被除数 ÷ 除数 ＝ 商
  • 零不能作为除数，因为零和任何数相乘所得的结果都等于零，所以作为除数是没有意义的。

运算顺序

  • 第一级运算：加减运算
  • 第二级运算：乘除运算
  • 第三级运算：乘方
  • 在没有括号的第一级运算或者第二级运算里，从左往右依次运算。
  • 在一个没有括号的算式里，如果既包含有第一级运算，也含有第二级运算，因该先做第二级运算，后做第一级运算。先乘除，后加减。
  • 一个算式里有括号的，括号里面的运算要想做。如果有几种括号，先算最里层的小括号里面的运行，在算较外面的中括号里面的运算，最后才算最外面的大括号里面的运算。

数轴

  • 数轴是一条用来表示数的直线，它要有规定的正方向，原点和单位长度。
  • 数轴上表示某个数的点叫做这个数的对应点，所有不想等的数都有不同的对应点。

相反数

  • 零的相反数就是这个数零本身。
  • 要表示一个数的相反的数，只要在这个数前面添上一个“－”号；如果这个数前面原来有正负号，要先添上括号后再在括号前面添上“－”号。
  • 两个相反数的绝对值是相等的。

有理数的乘方

  • 乘相同因数的积的运算叫做乘方，乘方的结果叫做幂。在aⁿ中，a叫做底数，n叫做指数，aⁿ读作a的n次方，把aⁿ看作a的n次方的结果时，也可以读作a的n次幂。

代数式

代数式

  • 用有限次的加、减、乘、除、乘方等运算符号把数字、字母连接起来的式子叫做代数式。
  • 用数字或者字母表示的一个单独的数，也可以看作是代数式。

代数式的值

  • 代数式是表示数的计算式子，如果代数式里的字母用指定的数去代替，再依照代数式里所表示的运算进行计算，所得的结果就叫做代数式的值。

整式和分式

  • 整式：有理式（有理代数式）里，没有除式，或者除式里没有字母(a+b)h/2
  • 分式：有理式里，不但有除法，而且除式里有字母出现1/a

Swift学习笔记（一）

发表于 2016-04-16   |   分类于 Programming Language
  1. 浮点数存在误差，不能用于精确计算（金额计算等）
  2. Swfit不会自动类型转换，不同数据类型（如Int与Int8）需明确转换成同类数据类型才能进行计算。
  3. switch找到匹配的case并执行该case的body后就会跳出switch，无需使用break。在匹配的case的body中可以使用fallthrough执行紧邻的下一个case的body而无需检查条件。
  4. 元组（tuple）是一个数据组合体，该组合体被当作一个数据看待。
  5. 字符串由一串character构成，每个character与一个统一码（Unicode）对应。有些character除了与其对应的统一码外，可以由其他两个统一码组合而成，swift视这两个character相同（==）。
  6. Stirng，Array，Dictionary是值类型（value type）。
  7. optional：String与String?是两个不同的数据类型。
  8. 函数有内部参数名（internal parameter name）和外部参数名（external parameter name）。
  9. in-out参数不可以有默认值。variadic参数不可以作为in-out参数。
  10. 函数也是一种数据类型，由参数和返回值构成。
  11. 数值类型（value type）不允许自身的方法改变自身（self），需借助mutating关键字。
  12. free functions == global functions
  13. lazy修饰的变量只会计算一次。
  14. 一个常量的类型如果是struct或者是enum（value type），他们的property即便是变量也不能修改。
  15. extension不支持添加存储属性（stored property）， 支持添加计算属性（computed property）。
  16. Objective-C所有的类都继承自根类（root class）NSObject，Swfit没有root class。
  17. Property observers are not triggered when the property value is change from within an initializer.
  18. Setting the active property on a constraint is preferable to calling addConstraint(_:) or removeConstraint(_:) youself.
